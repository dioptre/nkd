Install and Uninstall.

 

Installing and uninstalling mailserver.

 

Running as services (NT/2000/XP):

  • Just copy mailserver files from Release folder to some location in HDD.
  • Configure domains, users and settings.
  • Use register.bat to install server and a_unregister.bat to uninstall server.

 

Running as Tray app (WIN98/NT/2000/XP):

  • Just copy mailserver files from Release folder to some location in HDD.
  • Configure domains, users and settings.
  • Add new shortcut to Windows Startup menu as '...\lsMailServer.exe trayapp' and restart Windows. Trayapp parameter in lsMailServer.exe forces server to run as TrayApp.

 

Note: Running as Tray app is advisable to use only in Windows 98, because in order to use mailserver, you must be logged on in Windows.
