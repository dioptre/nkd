Security.

 

Table:

  • Description - Security entry description.
  • Protocol - Security entry protocol.
  • Type - Security entry type.
  • Action - Security entry action.
  • Content - Security entry content.

 

ToolBar Buttons:

  • Create New Security Entry - To create new security entry. Location - first button from left; appearance - white blank page.
  • Delete Security Entry - To delete selected security entry. Location - second button from left; appearance - with red stripes crossed out page.
  • Edit Security Entry - To edit selected security entry. Location - third button from left; appearance - page with small cross at upper right corner.

 

 

Create New Security Entry:

  • To create new security entry, click 'Create New Security Entry' button from toolbar, which opens security entry window.
  • Fill necessary fields.
  • To save data, click 'OK' button, else click 'Cancel' button.

 

Delete Security Entry:

  • Select security entry you want to delete from the table.
  • To delete security entry, click 'Delete Security Entry' button from toolbar.
  • After clicking 'Delete...' button appears window, which asks confirmation to delete security entry.
  • To delete security entry, click 'Yes' button, else click 'No' button.

 

Edit Security Entry:

  • Select security entry you want to edit from the table.
  • To edit security entry, click 'Edit Security Entry' button from toolbar, which opens security entry window.
  • Make necessary changes.
  • To save data, click 'OK' button, else click 'Cancel' button.
