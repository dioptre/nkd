Getting Started.

 

  • To configure server run MailServerManager.exe, which opens LS Mail Server Manager main window.

 

 

  • Make domain. Remove example row if needed. For window fields details see domains.

 

 

  • Make users. Remove example row if needed. For window fields details see users.

 

 

  • Configure security if needed. For window fields details see security.

    PS! If you do not know how to configure security, DO NOT REMOVE example rows. With those example rows you can send and receive mails and prevent others from using your mailserver to send mails.
