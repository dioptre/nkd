Routing.

 

Table:

  • Pattern - Route pattern.
  • Mailbox - Route mailbox.
  • Domain - Route domain.
  • Description - Route description.

 

ToolBar Buttons:

  • Create New Route - To create new route. Location - first button from left; appearance - white blank page.
  • Delete Route - To delete selected route. Location - second button from left; appearance - with red stripes crossed out page.
  • Edit Route - To edit selected route. Location - third button from left; appearance - page with small cross at upper right corner.

 

 

Create New Route:

  • To create new route, click 'Create New Route' button from toolbar, which opens route window.
  • Fill necessary fields.
  • To save data, click 'OK' button, else click 'Cancel' button.

 

Delete Route:

  • Select route you want to delete from the table.
  • To delete route, click 'Delete Route' button from toolbar.
  • After clicking 'Delete...' button appears window, which asks confirmation to delete route.
  • To delete route, click 'Yes' button, else click 'No' button.

 

Edit Route:

  • Select route you want to edit from the table.
  • To edit route, click 'Edit Route' button from toolbar, which opens route window.
  • Make necessary changes.
  • To save data, click 'OK' button, else click 'Cancel' button.
