Aliases.

 

Table:

  • Alias Name - Alias name.
  • Description - Alias description.
  • Members - Alias members.

 

ToolBar Buttons:

  • Create New Alias - To create new alias. Location - first button from left; appearance - white blank page.
  • Delete Alias - To delete selected alias. Location - second button from left; appearance - with red stripes crossed out page.
  • Edit Alias - To edit selected alias. Location - third button from left; appearance - page with small cross at upper right corner.

 

 

Create New Alias:

  • To create new alias, click 'Create New Alias' button from toolbar, which opens alias window.
  • Fill necessary fields.
  • To save data, click 'OK' button, else click 'Cancel' button.

 

Delete Alias:

  • Select alias you want to delete from the table.
  • To delete alias, click 'Delete Alias' button from toolbar.
  • After clicking 'Delete...' button appears window, which asks confirmation to delete alias.
  • To delete alias, click 'Yes' button, else click 'No' button.

 

Edit Alias:

  • Select alias you want to edit from the table.
  • To edit alias, click 'Edit Alias' button from toolbar, which opens alias window.
  • Make necessary changes.
  • To save data, click 'OK' button, else click 'Cancel' button.
