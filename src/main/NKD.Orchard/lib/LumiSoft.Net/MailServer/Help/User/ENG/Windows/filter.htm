Filter.

 

  • Description - Filter description.
  • Assembly - Filter assembly.
  • Get Assembly button - To get filter assembly.
  • Class - Filter class.
  • Cost - Filter cost.
  • Enabled - If checkbox is checked, then filter is enabled.
  • OK button - To save filter data.
  • Cancel button - Not to save filter data.
