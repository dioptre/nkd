Filters.

 

Table:

  • Cost - Filter cost.
  • Description - Filter description.
  • Assembly - Filter assembly.
  • Class Name - Filter class name.

 

ToolBar Buttons:

  • Create New Filter - To create new filter. Location - first button from left; appearance - white blank page.
  • Delete Filter - To delete selected filter. Location - second button from left; appearance - with red stripes crossed out page.
  • Edit Filter - To edit selected filter. Location - third button from left; appearance - page with small cross at upper right corner.

 

 

Create New Filter:

  • To create new filter, click 'Create New Filter' button from toolbar, which opens filter window.
  • Fill necessary fields.
  • To save data, click 'OK' button, else click 'Cancel' button.

 

Delete Filter:

  • Select filter you want to delete from the table.
  • To delete filter, click 'Delete Filter' button from toolbar.
  • After clicking 'Delete...' button appears window, which asks confirmation to delete filter.
  • To delete filter, click 'Yes' button, else click 'No' button.

 

Edit Filter:

  • Select filter you want to edit from the table.
  • To edit filter, click 'Edit Filter' button from toolbar, which opens filter window.
  • Make necessary changes.
  • To save data, click 'OK' button, else click 'Cancel' button.
