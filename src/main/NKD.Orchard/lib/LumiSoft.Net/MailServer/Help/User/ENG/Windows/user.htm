User.

 

General (picture 1):

  • Full Name - User full name.
  • User Name - User login name.
  • Password - User password.
  • Generate Password button - To generate user password.
  • Description - User description.
  • Max. Mailbox Size - Maximum mailbox size in megabytes.
  • User Enabled - If checkbox is checked, then user is enabled.
  • Allow Relay - If checkbox is checked, then relay is allowed.
  • E-mail Address - User e-mail address.
  • Domains - Domains.
  • E-mail Address list - User e-mail addresses.
  • Add button - To add user e-mail address.
  • Remove button - To remove selected e-mail address.

 

POP3 Remote Accounts (picture 2):

  • Server - POP3 server name.
  • Port - POP3 server port number.
  • User Name - User login name.
  • Password - User password.
  • Add button - To add account.
  • Delete button - To delete selected account.
  • Table - Accounts list.
