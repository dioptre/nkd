Users.

 

Table:

  • Mailbox - User mailbox.
  • E-mail Address - User e-mail address.
  • Description - User description.

 

ToolBar Buttons:

  • Create New User - To create new user. Location - first button from left; appearance - white blank page.
  • Delete User - To delete selected user. Location - second button from left; appearance - with red stripes crossed out page.
  • Edit User - To edit selected user. Location - third button from left; appearance - page with small cross at upper right corner.

 

 

Create New User:

  • To create new user, click 'Create New User' button from toolbar, which opens user window.
  • Fill necessary fields.
  • To save data, click 'OK' button, else click 'Cancel' button.

 

Delete User:

  • Select user you want to delete from the table.
  • To delete user, click 'Delete User' button from toolbar.
  • After clicking 'Delete...' button appears window, which asks confirmation to delete user.
  • To delete user, click 'Yes' button, else click 'No' button.

 

Edit User:

  • Select user you want to edit from the table.
  • To edit user, click 'Edit User' button from toolbar, which opens user window.
  • Make necessary changes.
  • To save data, click 'OK' button, else click 'Cancel' button.
