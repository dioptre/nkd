Domains.

 

Table:

  • Domain Name - Domain name.
  • Description - Domain description.

 

ToolBar Buttons:

  • Create New Domain - To create new domain. Location - first button from left; appearance - white blank page.
  • Delete Domain - To delete selected domain. Location - second button from left; appearance - with red stripes crossed out page.

 

 

Create New Domain:

  • To create new domain, click 'Create New Domain' button from toolbar, which opens domain window.
  • Fill necessary fields.
  • To save data, click 'OK' button, else click 'Cancel' button.

 

Delete Domain:

  • Select domain you want to delete from the table.
  • To delete domain, click 'Delete Domain' button from toolbar.
  • After clicking 'Delete...' button appears window, which asks confirmation to delete domain.
  • To delete domain, click 'Yes' button, else click 'No' button.
