System.

 

General (picture 1):

  • DataBase Type - Database type.
    • XML -
    • MSSQL -
  • Connection String - Database connection string.
  • Mail Store Path - Mail store folder location.
  • Log SMTP Commands - If checkbox is checked, then SMTP commands are logged.
  • Log POP3 Commands - If checkbox is checked, then POP3 commands are logged.
  • Log Server Events - If checkbox is checked, then server events are logged.
  • Path - Log file folder location.
  • Get Path button - To get log file folder location.

 

SMTP (picture 2):

  • Session Idle Timeout - Session idle timeout in seconds.
  • Command Idle Timeout - Command idle timeout in seconds.
  • Postmaster User Name - Postmaster login name.
  • Postmaster Password - Postmaster password.
  • Max. Message Size - Maximum message size in megabytes.
  • Max. Recipients per Message - Maximum recipients per message.
  • Max. Bad Commands - Maximum bad commands.
  • SMTP IP Address - SMTP IP address.
  • SMTP Port on IP Address Above - SMTP port number on IP address.
  • Max. SMTP Threads - Maximum SMTP threads.

 

POP3 (picture 3):

  • Session Idle Timeout - Session idle timeout in seconds.
  • Command Idle Timeout - Command idle timeout in seconds.
  • Max. Bad Commands - Maximum bad commands.
  • POP3 IP Address - POP3 IP address.
  • POP3 Port on IP Address Above - POP3 port number on IP address.
  • Max. POP3 Threads - Maximum POP3 threads.

 

Delivery (picture 4):

  • Send Mails Through Smart Host - Send mails through Smart Host.
    • Smart Host - Smart Host name.
  • Send Mails Directly using DNS - Send mails directly using DNS.
    • Primary DNS - Primary DNS address.
    • Secondary DNS - Secondary DNS address.
  • Max. Delivery Threads - Maximum delivery threads.
  • Relay Interval - Relay interval in seconds.
  • Relay Retry Interval - Relay retry interval in seconds.
  • Send Undelivered Warning After - Undelivered warning text sending in minutes.
  • Edit Undelivered Warning Text button - To edit undelivered warning text
  • Send Undelivered After - Undelivered text sending in hours.
  • Edit Undelivered Text button - To edit undelivered text.
  • Store Undelivered messages - If checkbox is checked, then undelivered messages are stored.
